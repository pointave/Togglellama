## ToggleLlama (Windows)

System tray app to handle llama-server routing and embedding. On first setup you will enter in filepaths for llama.cpp build, model directory, and embedding model. You'll then have in your system tray the Llama.cpp Icon. 

Double click on icon to start server, double click then converts to an unload model function. There is a configuration menu to change various settings and add flags to your command.

After the server has been started, new bat files are created if you like the old method of starting and unloading models.

***Configuration menu is weird, usually cancel and save works but if it is unresponsive move the window to restore its focus.***
