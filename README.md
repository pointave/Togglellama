## ToggleLlama (Windows)

LLama.cpp finally allows model swapping! The command we are running launches llama.cpp server and it you access the web-interface you can toggle indiviual models. It works amazingly already but its kind of slow to unload and you have to navigate and scroll to the model you have loaded, so I made another toggle to quickly kill or relaunch the server. 

Ollama added their own web-ui recently too which kind of made its toggle annoying pulling up that window whenever relaunching, but sometimes thats where you plan on going so... The model list bat is just a terminal for ollama models but superflouous now.
